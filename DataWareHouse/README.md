#Data Warehouse
Construcción de un Data Warehouse, definiendo dimensiones y tabla de hechos. Para el analisis del tema Homicidios en Colombia, tratado en clase.

Scripts: Contiene los script SQL de las tablas que almacenarán los datos extraídos, transformados y cargados desde el Data Lake, creado previamente.
Modelo ER: Modelo Entidad Relación de Copo de Nieve del Data Warehouse
